# Open-world route tuning pass 1 (PR 3F)

This pass follows the PR 3E route readiness report layer and intentionally tunes only:

- `route_westwild`
- `route_south_coast_stub`

## Route focus

- `route_westwild` is treated as the first early combat route tuning target.
- `route_south_coast_stub` remains a sparse/utility stub route.

## Readiness intent

- Westwild should remain a non-stub route with solo, pack, and elite-anchor presence.
- South Coast should remain sparse/stub and should not be forced into full combat-route completeness warnings.
- Rare anchors for Westwild may remain deferred when not represented in current live route-local data.

## Explicit non-goals

- no reward number changes
- no reward formula changes
- no combat formula changes
- no new mobs or new content surfaces
- no route topology changes
- no blanket skill rollout
- mixed-mob packs remain future work
